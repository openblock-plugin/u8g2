# U8G2

![Version](https://img.shields.io/github/v/tag/openblock-plugin/u8g2?label=version)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Type](https://img.shields.io/badge/type-extension-orange.svg)

U8g2 is a general graphics library for multiple screens, which is suitable for LCD (base on ST7920) or OLED (base on SSD1306) screens.
